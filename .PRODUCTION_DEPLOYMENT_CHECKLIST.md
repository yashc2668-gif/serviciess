**Location**: Railway Dashboard → Backend Project → Variables

**Add these variables**:

| Name | Value | Notes |

| `DEBUG` | `False` | Disables debug mode in FastAPI |
| `ALLOWED_ORIGINS` | `["https://m2n-frontend-git-main-yashs-projects-8e52d41e.vercel.app","https://m2n-frontend-jel4ixehf-yashs-projects-8e52d41e.vercel.app"]` | Allow both Vercel domains for CORS |
| `SECRET_KEY` | [Use existing or generate new] | Keep existing Railway secret if available |
| `POSTGRES_PASSWORD` | [Use existing Railway Postgres password] | Keep existing |
